# 5½u Macropad

## 📻 Project Description

A macropad inspired by the look of the iconic beam spring keyboards of the '70s.

![Photo of the macro pad](images/photos/20231022-IMG_1909.jpg)

## 🧰 Features

- 12× keys
- RP2040 microcontroller
- Powered by QMK with Vial support
- USB-C connector
